# Pareto Active Learning with Gaussian Processes and Adaptive Discretization
This repository contains the implementation of the Adaptive ePAL algorithm from the paper ["Pareto Active Learning with Gaussian Processes and Adaptive Discretization"](https://arxiv.org/abs/2006.14061). 
To initiate an experiment, set the "func_list" and "kernel_list" parameters in "main.py" file. Function names should match the file names in the "example_setups" directory. The results of an experiment are saved in "Results" directory. 

## Required Packages
- GPy
- h5py
